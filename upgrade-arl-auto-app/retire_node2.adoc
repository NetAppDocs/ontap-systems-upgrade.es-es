= Retire el nodo 2
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Para retirar el nodo 2, apáguelo correctamente y luego retírelo del rack o chasis.

.Pasos
. Reanude la operación:
+
`system controller replace resume`

+
El nodo se detiene automáticamente.

+
.. Opcionalmente, decomisionar el nodo2 una vez completada la actualización. Ver link:decommission_old_system.html["Retire el sistema antiguo"].




.El futuro
Tras retirar el nodo2, link:install_boot_node4.html["instalar y arrancar node4"].
