[NOTE]
====
* Si vas a actualizar un sistema intercambiando módulos de controladora en el mismo chasis, como AFF A800 o AFF C800, NetApp recomienda encarecidamente usar el procedimiento de actualización que link:../upgrade-arl-auto-in-chassis/index.html["actualiza los modelos de controladores usando ARL, manteniendo el chasis del sistema, los discos y los datos existentes"]. Este procedimiento ARL incluye los pasos que aseguran que los discos internos permanezcan seguros en el chasis cuando quites e instales las controladoras durante el procedimiento de actualización.
+
link:../upgrade-arl-auto-in-chassis/decide-to-use-the-aggregate-relocation-guide.html#supported-systems-in-chassis["Conoce las combinaciones de actualización de sistemas compatibles usando ARL, manteniendo el chasis, los discos y los datos del sistema existente"].

* Puede usar el cifrado de almacenamiento de NetApp (NSE), el cifrado de volúmenes de NetApp (NVE) y el cifrado de agregados de NetApp (NAE) con este procedimiento.
